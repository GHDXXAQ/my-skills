# My Skills

记录 AI 学习过程中产出的 Claude Code 技能（Skills）集合。

## 项目简介

这是一个持续积累的 Claude Code 技能仓库，存放我在学习 AI 过程中编写和打磨的实用技能。每个技能都是独立的模块，可直接加载到 Claude Code 中使用。

## 技能列表

### 🧠 冷知识向导（knowledge）

一款每天自动推送冷知识的趣味技能。每天 21:00 自动推送一条引人好奇的冷知识，故事型科普风格，像朋友讲故事一样娓娓道来。

**核心能力：**
- **每日推送** — 自动定时触发，每天一条不重样
- **用户画像适配** — 根据你的职业和兴趣方向，推送更对你胃口的冷知识
- **36 大知识领域** — 覆盖物理、天文、生物、历史、音乐、哲学等
- **30 天去重** — 一个月内不会重复推送相同内容

**使用方式：** 说"来条冷知识"或等待每日 21:00 自动推送

### 📐 金字塔原理（pyramid-principle）

一套基于金字塔原理的结构化思维与高效沟通工具。帮你把混乱的想法变成清晰的金字塔结构。

**核心能力：**
- **内容诊断** — 分析现有内容是否符合金字塔原理，给出评分和改进建议
- **结构重构** — 将杂乱无章的内容重新组织为金字塔结构
- **从零构建** — 用 SCQA 框架帮你搭建报告、提案或演讲的完整结构
- **MECE 分析** — 对复杂问题进行不重叠、不遗漏的全面分解

**使用方式：** 说"帮我分析这段文字的结构"或"按金字塔原理重新组织"

## 快速开始

### 环境要求

- Claude Code（用于加载和运行技能）
- Python 3.8+（部分技能脚本需要）

### 安装

将技能目录配置到 Claude Code 的 skills 路径即可使用：

```bash
# 将 skills 目录链接或复制到 Claude Code 的技能目录
cp -r skills/* ~/.claude/skills/
```

### 基本用法

在 Claude Code 中直接输入对应的触发词即可使用各技能：

```text
"来条冷知识"              → 调用冷知识向导
"分析这段文字的结构"      → 调用金字塔原理
"按金字塔原理重新组织"    → 调用结构重构
"用 MECE 分析这个问题"   → 调用 MECE 分析
```
