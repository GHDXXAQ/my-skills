# 金字塔原理核心概念

## 1. 金字塔结构

```
        结论 / 中心思想
        ┌─────────────────┐
        │   中心论点       │  ← 顶层（1个）
        └────────┬────────┘
       ┌─────────┼─────────┐
   ┌───┴───┐ ┌───┴───┐ ┌───┴───┐
   │ 论据A  │ │ 论据B  │ │ 论据C  │  ← 中层（3-7个）
   └───┬───┘ └───┬───┘ └───┬───┘
   ┌┴┐ ┌┴┐ ┌┴┐ ┌┴┐ ┌┴┐ ┌┴┐
   │ │ │ │ │ │ │ │ │ │ │ │    ← 底层（支撑证据）
```

**核心规则：**
- **结论先行**：每个层级必须先给出结论/概括
- **以上统下**：上层是对下层的概括，下层是对上层的支撑
- **归类分组**：同一层级的论点属于同一逻辑范畴
- **逻辑递进**：同一层级按某种逻辑顺序排列（时间/结构/程度）

## 2. 纵向关系（Vertical Relationship）

上层是下层的**概括/结论**，下层是上层的**解释/支撑**。

两种交互模式：
1. **问-答式**：读者看到上层论点 → 问"为什么" → 下层给出答案
2. **演-推式**：上层是推理结论，下层展示推理过程

> 检查方法：每一层读完后，自问"读者会问什么问题？"下一层是否回答了这些问题？

## 3. 横向关系（Horizontal Relationship）

同一层级的论点之间必须符合逻辑关系：

### 演绎推理（Deductive）
三段论：大前提 → 小前提 → 结论
```
所有人都会死         ← 大前提
苏格拉底是人         ← 小前提
所以苏格拉底会死     ← 结论
```
**适用**：需要说服性强、逻辑严密的论证

### 归纳推理（Inductive）
将多个相似的事实/观点归为一类，提炼共同结论
```
A公司降本增效
B公司降本增效        ← 归纳
C公司降本增效             ↓
所以在当前环境下降本增效是明智的
```
**适用**：分析问题、分组归类

## 4. MECE 原则

**Mutually Exclusive, Collectively Exhaustive**
- **相互独立**：每个分组内的子项不重叠
- **完全穷尽**：所有子项覆盖了母项的全部范围

### 常用 MECE 框架

| 框架 | 应用场景 |
|------|----------|
| 3C (Company/Customer/Competitor) | 商业分析 |
| 4P (Product/Price/Place/Promotion) | 营销策略 |
| 人/机/料/法/环 | 制造业问题分析 |
| 外部因素/内部因素 | 战略分析 |
| 过去/现在/未来 | 时间维度分析 |
| 宏观/中观/微观 | 多层级分析 |

### MECE 检查清单
- [ ] 同一层的子项之间是否有重叠？
- [ ] 所有子项加在一起是否覆盖了母项的全部？
- [ ] 分类维度是否一致？（不要混用不同分类维度）
- [ ] 每个子项的粒度是否一致？

## 5. SCQA 框架

用于**开场/序言**的结构化表达框架：

```
S (Situation)  →  背景：读者已知且认同的事实
C (Complication) → 冲突：发生了什么变化/问题
Q (Question)    → 问题：由此产生的关键问题
A (Answer)      → 答案：你的核心观点/解决方案
```

### SCQA 变体

| 变体 | 结构 | 适用场景 |
|------|------|----------|
| 标准式 | S-C-Q-A | 正式报告、提案 |
| 开门见山式 | A-S-C-Q | 结论先行，适合忙碌决策者 |
| 突出忧虑式 | C-S-Q-A | 从痛点切入，制造紧迫感 |
| 对比式 | C-A-S-Q | 突出方案差异，适合竞争分析 |

## 6. 金字塔的构建方法

### 自上而下法（推荐）
1. 确定中心思想（结论/核心主张）
2. 列出读者会问的关键问题
3. 用 MECE 分组回答每个问题
4. 继续向下分解（重复 2-3）

### 自下而上法
1. 列出所有要点/事实
2. 将相关要点分组归类
3. 提炼每组的主旨概括
4. 继续向上提炼，直至单一结论

## 7. 常见错误

1. **结论不明确**：说了半天没有核心观点
2. **层次混乱**：把不同层级的论点放在同一层
3. **MECE 不满足**：子项重叠或遗漏
4. **逻辑跳跃**：下层不能充分支撑上层
5. **结构失衡**：某个分组异常庞大而其他分组过简
6. **分类维度不一致**：混用了时间维度和结构维度
7. **缺乏递进**：同级论点之间毫无逻辑顺序
